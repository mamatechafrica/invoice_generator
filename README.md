**100% Free & Open Source** — no sign-ups, no subscriptions, no hidden fees. Ever.

A simple, professional invoice generator that runs entirely in your browser. Your data never leaves your computer.

## Features

- Multiple currencies (USD, KES, GBP, EUR)
- Client management — save & reuse client details
- Auto-calculating invoice amounts (enter total + percentage, amount calculates automatically)
- Smart invoice numbering (custom prefix + year + sequence: `INV-2026-0001`)
- Print to PDF or download as HTML
- Email invoice directly from the app
- Works offline

## Download & Use

### Use Online
Visit **[mamatechafrica.github.io/invoice_generator](https://mamatechafrica.github.io/invoice_generator)**

### Download for Offline Use
1. Click **Code → Download ZIP** above
2. Unzip the folder anywhere on your computer
3. Open `index.html` in any browser
4. Start creating invoices — no internet needed

### Clone the Repo
```bash
git clone https://github.com/mamatechafrica/invoice_generator.git
cd invoice_generator
open index.html
```

## Usage

1. Fill in your details under **Account Settings**
2. Click **Save Account Info**
3. Add client details and invoice information
4. Click **Generate Invoice** to preview
5. **Print/PDF** to save, **Download** as HTML, or **Email** to send

## Privacy

All data is stored **only in your browser** using localStorage. Nothing is sent to any server. Your information remains private on your device.

## Deploy Your Own

Host it for free on GitHub Pages:

```bash
git clone https://github.com/mamatechafrica/invoice_generator.git
```
Then go to your repo **Settings → Pages** and set the source to `main`.

# Contributing to Invoice Generator

Thank you for your interest in contributing! This is a free and open source project, and contributions of all kinds are welcome.

## Ways to Contribute

### Report Bugs
- Use the **Bug Report** template when opening an issue
- Include steps to reproduce, expected behaviour, and your browser/device info
- Screenshots are extremely helpful

### Suggest Features
- Use the **Feature Request** template
- Explain the problem your feature would solve
- Keep it simple — this is a lightweight browser-based tool

### Submit Code
1. **Fork** the repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/invoice_generator.git
   ```
3. **Create a branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes** — keep them focused and clean
5. **Test** by opening `index.html` in your browser
6. **Commit** with a clear message:
   ```bash
   git commit -m "Add: brief description of your change"
   ```
7. **Push** and open a **Pull Request**

### Code Guidelines
- This is a single-file HTML app — keep it that way if possible
- Follow the existing code style (2-space indent, no semicolons in JS)
- No external dependencies unless absolutely necessary
- All data stays in localStorage — never add server-side storage or tracking
- Test on major browsers: Chrome, Firefox, Safari, Edge

### Spread the Word
- Star the repo
- Share with colleagues
- Fork and customize for your own needs

## Questions?

Open a discussion or issue — we're happy to help!
